# Game Design Document
## Game Concept
It's a "Simon" memory game. 4 images will be displayed and flash in a specific order. The player clicks the images in the same order to get points. Upon each success the flash count is incremented.

## Target Platforms

* Cross-platform: desktop and mobile
* Browser Game

## UI / Player Controls

* Mouse click / touch screen
* When an image is clicked, it has a success or failure animation
